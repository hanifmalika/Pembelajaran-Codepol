Make a Change

1. Git add (nama file) dan Git add.-git status untuk melihat status file yang sudah di add atau di reset.

2. Git reset untuk membatalkan file yang dimasukan dari git add.

3. Git commit untuk menyimpan perubahan ke dalam repository lokal.

git commit -m "Melakukan agar lebih jelas"

Observe

4. git diff untuk melihat perubahan kalau sudah di add atau di commit

git status-------git show------git log

SYNC----Masuk ke Website Github

5. Git push untuk mengirim perubahan ke remote repository

- git remote add origin [url] ----- git remote -v ---------git push origin master

6. Git pull origin master untuk mengambil perubahan dari remote repository, disini saya mengambl about.html

7. git fetch ----- git checkout [branch_name], Untuk mengganti ke branch tertentu yang telah kita buat di website repository github

Branch

8. git checkout Fitur-A ------- git branch --- untuk melihat branch kita berada dan melihat branch yang tersedia

9. git branch -D Fitur-A -- menghapus branch Fitur-A

10. git brach [nama brach yang mau dibuat]

git branch fitur-contact--------git checkout fitur-contact-----membuat file bernama contact-us.html------melakukan git commit -m "Menambahkan halaman contact"----- Pindah ke master dengan-- git checkout master

disini saat dimaster file contact us tidak terlihat, kita harus kembali ke brach fitur-contact untuk melihat file contact-us.html

11. git merge -- Untuk menggabungkan file dari branch fitur-contact ke branch master
